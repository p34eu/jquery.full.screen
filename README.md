# jquery.fullscreen

jquery fullscreen plugin
Full screen any element. By default toggles the web page.



#Usage:

     $('[data-toggle="fullscreen"]').fullscreen();


#Options
     iconEnter:
     iconExit:

#Callbacks

     onenter
     onexit


#Install

    bower install jquery.full.screen
